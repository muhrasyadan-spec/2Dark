# Sistem Pendataan Penjual Minuman

Aplikasi sederhana berbasis HTML, CSS, dan JavaScript.

## Fitur
- Tambah data penjual
- Simpan data ke LocalStorage
- Hapus data
- Tampilan responsif

## Cara Menjalankan
1. Download project
2. Buka index.html

## Deploy ke GitHub Pages
1. Upload ke repository GitHub
2. Masuk Settings
3. Pilih Pages
4. Deploy dari branch main
5. Aplikasi akan online
